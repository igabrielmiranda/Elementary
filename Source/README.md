# Source

Esta pasta agora concentra a source/build separada do runtime.

- `Source/Server/`: `src`, `vc17`, `cmake`, `CMakeLists.txt` e arquivos de desenvolvimento do servidor
- `Source/Cliente/`: `src`, `vc16`, `android`, `tests`, `tools`, `CMakeLists.txt` e arquivos de desenvolvimento do client

Os runtimes continuam em `Server/` e `Cliente/`, com working directory preservado.
